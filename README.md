# bionic-chrome-extension
Convert text on webpage to bionic extension
Highlight the text and opens in a new tag
Demo - https://screenshot.click/19-19-kfoth-jydpj.mp4

Just a proof of concept, not the best way to do any of this. 
Visit Rapid API and sign up to get your free token - https://rapidapi.com/bionic-reading-bionic-reading-default/api/bionic-reading1/
Replace "X-RapidAPI-Key" value with your fresh new key
